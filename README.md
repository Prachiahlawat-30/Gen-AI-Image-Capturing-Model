# Gen AI Image Captioning Model

An AI-powered Image Captioning application built using the BLIP (Bootstrapping Language-Image Pretraining) model from Hugging Face and Gradio.

## Features

* Upload any image
* Generate human-like captions automatically
* Interactive web interface using Gradio
* Powered by Salesforce BLIP model
* Easy deployment and usage

## Tech Stack

* Python
* Gradio
* Hugging Face Transformers
* BLIP Model
* PIL (Python Imaging Library)

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Gen-AI-Image-Capturing-Model.git
cd Gen-AI-Image-Capturing-Model
```

Create a virtual environment:

```bash
python -m venv genai_env
```

Activate the environment:

Windows:

```bash
genai_env\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:7860
```
